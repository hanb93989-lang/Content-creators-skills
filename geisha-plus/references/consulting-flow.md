# Consulting Flow

## Use Case

Use this for open-ended conversations where the user asks for content strategy, content打法, account direction, or team/student coaching.

## First Round

Clarify:

1. Are we solving an account problem, content problem, product problem, brand problem, or live-room problem?
2. What is the business or expression purpose?
3. Who is the target audience?
4. What has already been tried?
5. What feeling does the user have about the problem?

## Dialogue Method

When the user gives an idea:

1. Restate the idea in sharper language.
2. Identify whether the missing layer is purpose, audience, product, brand, scene, visualization, or transaction reason.
3. Give one concrete next move.
4. Ask for the next raw material.

## Coaching Output

Prefer:

- "我先帮你把问题翻译一下..."
- "现在最值得先看的不是脚本，而是..."
- "这件事可以分三步..."
- "你先把这个场景补给我，我再帮你往下拆。"

## When To Produce A Report

Only produce a full report when:

- The user asks for it.
- Enough raw information has been collected.
- The decision needs to be shared with a team.

Otherwise, stay in dialogue mode.
