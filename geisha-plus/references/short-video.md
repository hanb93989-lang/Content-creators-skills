# Short Video

## First Judgment

Before writing a topic or script, ask:

- Who is this for?
- Why would this person stop?
- What scene makes the value visible?
- What emotion or transaction reason will make them continue?
- What does this topic do for the account or brand?

## Topic Upgrade Path

Move from event to judgment:

- Weak: "我去买绿植了"
- Stronger: "我想让家里长出一点生命力"
- Strongest: "让一个真正懂空间氛围的人，帮我判断这个家该长出什么"

## Hook Criteria

A good opening should contain at least one of:

- A visible contradiction.
- A clear user pain.
- A desirable result.
- A specific scene.
- A surprising judgment.
- A recognizable brand or object.

## Structure

Use this structure when the user only has a rough idea:

1. Evidence first: show why this topic matters.
2. Judgment tension: name the wrong way most people think about it.
3. Process: show selection, comparison, or decision.
4. Turn: reveal a corrected understanding.
5. Close: leave a memorable sentence or image.

## E-Commerce Material Structure

When the user is making paid traffic, Douyin e-commerce, product seeding, or挂车短视频, diagnose through:

1. Opening: pain, question,反向推荐, conflict, expert trust, or visual surprise.
2. Product use: show the product in a real scenario.
3. Detail proof: show material, texture, size, sound, smell, touch, durability, or freshness.
4. Result: show user benefit or before/after.
5. Transaction reason: price, service, urgency, trust, scenario need, or social gift logic.

Common shooting formulas:

- 原生人物否定开场 + 产品使用 + 效果展示
- 问题开头 + BGM/画面卡点 + 产品展示 + 情绪价值
- 黄金3秒开场 + 产品介绍 + 细节展示 + 材质溯源 + 服务/转化理由
- 夸张痛点开头 + 适用场景 + 细节使用 + 产品介绍 + 引导转化
- 专家强化信任 + 制造焦虑 + 科普 + 使用场景 + 多产品展示
- 争吵互动开头 + 解压/加量信息 + 数量展示 + 价格引导

## Short-Video Diagnosis

If a short-video idea feels ordinary, test whether it lacks:

- A precise audience.
- A visible scene.
- A judgment difference.
- A specific object or behavior.
- A reason for the audience to save, comment, or buy.
- A link to the account's long-term positioning.
- A platform-quality baseline: clear sound, stable volume, clear picture, readable language/subtitles, and enough interaction potential.
